# My Shell

I made it for studying System Programming & C

<br>

## Implementated Features
- `cd`
- `history`
- Multiple Commands (`;`)
- Background execution (`&`)
- Redirection `>`, `>>`, `<`
- Redirection `>|` (Ignore `set -o noclobber`)
- Pipe `|`
- Multiple Pipe

> \+ `set -o noclobber`, `set +o noclobber`
